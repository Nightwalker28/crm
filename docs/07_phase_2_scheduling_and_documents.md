# Phase 2: Scheduling and Document Readiness

## Goal

Improve customer-facing sales readiness without adding campaign tracking or public lead-capture forms.

## What this phase changes

- Prospects can book meetings through public booking links.
- Calendar booking links gain availability windows, intake questions, and CRM event creation.
- Documents gain version history and reusable templates.

## Included task files

1. `08_task_calendar_booking_links.md`
2. `09_task_documents_versioning_templates.md`

## Acceptance criteria for the whole phase

- Public booking link can create calendar events.
- Public slots respect configured availability and existing booked events.
- Documents can preserve version history.
- Document templates can be reused.
- No campaign tracking, marketing automation, or public lead-capture form module is added.

## Excluded

- AI-generated content
- Telephony
- Campaign tracking
- Public lead-capture forms
